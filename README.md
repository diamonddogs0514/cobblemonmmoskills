# Cobblemon MMO Skills

Cobblemon MMO Skills adds mcMMO-style skill progression to Minecraft Fabric servers with Cobblemon integration.

## Features

- Gathering, combat, and utility skill progression
- Cobblemon capture, breeding, and training progression
- In-game skill stats GUI
- Skill leaderboards
- Persistent player skill data

## Requirements

- Minecraft 1.21.1
- Fabric Loader 0.17.2 or later
- Fabric API
- Fabric Language Kotlin
- Architectury API
- Cobblemon 1.7.3 for Minecraft 1.21.1

## Building

Use the Gradle wrapper:

```powershell
.\gradlew.bat build
```

The built mod jar will be generated under `build/libs/`.
